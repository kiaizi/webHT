Instructions:

  1. 1. Put all the fields that you want to duplicate inside a <fieldset> tag.
  2. 2. Place the 'add-button' input directly below the fieldset it is responsible for.
  3. 3. Finally, put "submit" button below all of the <fieldset> tags, i.e. inside the <form> tag.

Example 1 jQuery:

	$(document).ready(function(){
	    var options = {
	        numTimes: 1,
	        maxFields: 3,
	        maxFieldsMsg: 'You have reached the maximum number of fields allowed',
			fadeDuration: 'slow',
	        deleteButtonDiv: 'delete-button-div',
			deleteButtonClass: 'delete-button',
			deleteButtonValue: 'Delete This Fieldset'
	    }
	    $('#addButton1').dynamicForm(options);
	});
